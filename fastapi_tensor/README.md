# FastAPI 프레임워크 API 사용방법

### 가상환경 생성
```sh
# FastAPI 상위폴더에서 생성 추천
python -m venv [venv_name]
cd [venv_name] #venv 폴더로 이동
.\Scripts\Activate.ps1 # Powershell인 경우 
```
### 가상환경에서 필요 라이브러리 설치
```python
pip install -r requirements.txt
```
### FastAPI 폴더로 이동하여 FastAPI 실행
```python
python ./main.py
```
